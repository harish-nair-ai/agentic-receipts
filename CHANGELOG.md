# Changelog

All notable changes to Receipts will be documented in this file.

## [0.1.0] - 2026-07-09

### Added
- Initial release 🎉
- Claim extraction from agent completion messages (regex + LLM fallback)
- Deterministic evidence matching against session transcripts
- Independent LLM judge for fuzzy claim verification (Gemini, OpenAI, Anthropic)
- Beautiful Rich terminal receipt cards with ✅/⚠️/❌ verdicts
- Claude Code Stop hook integration (`receipts install`)
- Session stats aggregation (`receipts stats`)
- Zero-config setup — just set an API key and go
- MIT license
